Three-column responsive layout

Mobile-first: stacks to a single column on small screens, expands to three columns on wider screens (≥768px).

Column 1

Content for the first column. Put any HTML here — lists, images, forms, whatever you need.

Column 2

Content for the second column. The layout will keep equal column widths on larger screens.

Column 3

Content for the third column. On mobile, these stack vertically in the same order.
